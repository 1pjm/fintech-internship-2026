## 3. 전처리 파이프라인에서의 AI 활용

### 3-1. 아키텍처 설계 자동화

9개 원본 데이터를 단일 파일로 병합하려는 초기 설계를 AI가 데이터 입도(Granularity) 분석을 통해 문제를 사전 식별한다. AI는 시계열 × 지역 × 업종 단위가 혼재되는 경우 카테시안 곱(Cartesian Product) 폭발이 발생함을 지적하고, 데이터 성격(시계열/정적, 지역 단위/점포 단위)에 따라 **4개 마스터 테이블 구조로 재설계**하는 아키텍처를 제안한다.

| 마스터 테이블 | 성격 | AI가 판단한 분리 이유 |
|--------------|------|---------------------|
| 지역\_관광\_거시지표 | 시계열 | 월별 경제지표와 분기 GRDP의 시점 불일치 → Outer Join 권고 |
| 상권\_관광\_소비지표 | 시계열 | 업종 키 불일치(대분류명 ↔ 업종) → Rename 후 병합 |
| 공간\_시각화\_통합 | 정적 | 좌표 데이터는 시계열과 병합 시 행 수 폭발 위험 |
| 관광\_정적지표\_통합 | 정적 | 유입유출비율은 누적 스냅샷, 연월 키 없음 |

### 3-2. 전처리 이슈별 코드 생성

전처리 과정에서 발생한 5종의 기술 이슈에 대해 AI가 원인 진단과 해결 코드를 동시에 제공한다.

**① 인코딩 충돌 해결**  
`utf-8` / `cp949` 혼재 및 `.xlsx` 파일을 `read_csv()`로 읽는 오류에 대해, 파일 확장자 분기 처리와 `encoding` 파라미터 명시 로직을 생성한다.

```python
# AI 생성 코드: 확장자 기반 자동 분기 로드
def load_file(path):
    if path.endswith('.xlsx'):
        return pd.read_excel(path, sheet_name='경남 100대 생활업종', skiprows=5)
    for enc in ['utf-8', 'cp949']:
        try:
            return pd.read_csv(path, encoding=enc, low_memory=False)
        except UnicodeDecodeError:
            continue
```

**② 날짜 포맷 표준화**  
`202404`, `2024-04-01`, 분기/반기 등 이종 포맷을 `YYYY-MM-01` 단일 표준으로 변환하는 `standardize_year_month()` 함수를 AI가 정규표현식 기반으로 생성한다.

**③ Mac/Windows 한글 파일명 NFD 분해 문제**  
구글 드라이브 Mac 환경에서 한글 파일명이 자모 분리되는 NFD 현상을 AI가 진단하고, `unicodedata.normalize('NFC')` 기반의 절대경로 탐색 함수 `get_file_path()`를 생성하여 OS 무관 파일 탐색을 구현한다.

**④ 병합 키 타입 불일치 해결**  
`구` 컬럼 결측(NaN)으로 인한 `float64` ↔ `object` 타입 충돌을 진단하고, 병합 전 `fillna('').astype(str)` 전처리 함수 `clean_merge_keys()`를 자동 생성한다.

### 4-3. 런타임 오류 자동 진단 및 수정

EDA 실행 중 발생한 오류에 대해 AI가 원인을 추적하고 수정 코드를 제공한다.

**오류 1 — matplotlib 한글 폰트 누락**  
`WARNING: Font family 'NanumGothic' not found` 반복 경고에 대해 AI가 matplotlib 폰트 캐시 파일 삭제 → `_load_fontmanager(try_read_cache=False)` 강제 재로드 → `addfont()` 경로 등록의 3단계 해결책을 제시한다.

**오류 2 — FixedLocator 불일치**  
`ValueError: The number of FixedLocator locations (6) does not match the number of labels (12)` 오류에 대해 AI가 데이터에 6개 월만 존재하는 상태에서 12개 월 레이블을 강제 설정하는 것이 원인임을 진단하고, `.reindex(range(1,13), fill_value=0)` 전처리를 추가한다.

**오류 3 — NaN 전파로 인한 빈 차트**  
`평균숙박일수` 컬럼이 LEFT JOIN 후 전체 NaN이 되어 `median()` = NaN → `fillna(NaN)` → 점수 전체 NaN → 차트 미렌더링 연쇄 발생을 AI가 데이터 흐름 추적으로 진단한다. `minmax_safe(series, fallback)` 함수를 설계하여 전체 NaN 시리즈에 도메인 지식 기반 대체값(경남 평균 숙박일수 2.7일)을 적용하고, `np.nanmin` / `np.nanmax`로 색상 배열을 안전하게 정규화한다.

```python
# AI 설계: NaN 안전 Min-Max 정규화
def minmax_safe(series, fallback=None):
    filled = series.copy()
    if filled.isna().all():
        filled = filled.fillna(fallback if fallback is not None else 0)
    else:
        filled = filled.fillna(filled.median())
    s_min, s_max = filled.min(), filled.max()
    if s_max == s_min:
        return pd.Series(0.5, index=series.index)
    return (filled - s_min) / (s_max - s_min)
```

## 5. CSI 산출식 AI 검증

### 5-1. 산출식 논리 오류 탐지

초기 CSI 산출식에 대해 AI가 다음 3가지 오류를 자동으로 탐지한다.

```
초기 산출식 (오류 포함):
CSI = 100 − (매출증감지수 × 0.40 + 업종집중도지수 × 0.25
           + 폐업증가지수 × 0.20 + 소비변동성지수 × 0.15)
```

**오류 ①: 매출증감지수 방향 역전**  
`100 −` 구조에서 "높을수록 위험"인 지수만 사용해야 하나, 매출이 증가(+)하면 CSI가 낮아지는 논리 역전이 발생한다. AI가 방향 반전 필요성을 지적하고 `매출감소지수`로의 대체를 권고한다.

**오류 ②: 폐업증가지수 데이터 부재**  
AI가 4개 파일 전체를 스캔하여 폐업 이력 컬럼이 존재하지 않음을 확인한다(가맹점 파일 = 현재 스냅샷만 포함). 카드매출\_2의 `당월_사업자수` / `전년동월_사업자수` 역시 전체 NaN임을 확인하고, 대체 지표 2종(가맹점 밀도 역수, 활성 업종 감소수)을 제안한다.

**오류 ③: 2023→2024 연간 비교 스케일 이상**  
AI가 2023 대비 2024 매출 증감률이 전 지역에서 +400%~+1300%로 비정상임을 탐지하고, 데이터 집계 단위 불일치를 원인으로 진단한다.

### 5-2. 분기별 추세 지수 설계 지원

연간 비교 불가 문제에 대한 대안으로 연도 내 분기별 선형 추세 기울기를 매출 지수로 활용하는 방법론을 AI가 설계한다.

```python
# AI 설계: 분기별 선형 추세 기울기 계산
def linreg_slope(x, y):
    x, y = np.array(x, float), np.array(y, float)
    xm, ym = x.mean(), y.mean()
    return np.sum((x - xm)*(y - ym)) / np.sum((x - xm)**2)

# 기울기 < 0 → 연중 지속 하락 → 위험 상권
slope_pct = slope / mean_sales * 100
매출추세지수 = MinMax(-slope_pct)  # 부호 반전: 하락할수록 지수 높아짐
```

AI가 소비변동성지수(CV)와의 개념 중복 여부를 상관분석(r = 0.445)으로 검증하고, 중복 임계선(|r| > 0.7) 미만임을 확인하여 병용 가능성을 수치로 입증한다.

### 5-3. 최종 수정 산출식

```
CSI = 100 − (매출추세지수   × 0.40
            + 업종집중도지수 × 0.25
            + 상권위축지수   × 0.20
            + 소비변동성지수  × 0.15)

· 매출추세지수:  2024년 분기별 매출 선형 기울기 부호 반전 후 MinMax [0,100]
· 업종집중도지수: HHI (카드매출_2 업종별 매출 비중 제곱합) MinMax [0,100]
· 상권위축지수:  가맹점 밀도 역수 또는 활성 업종 감소수 MinMax [0,100]
· 소비변동성지수: 2024년 월별 매출 변동계수(CV = σ/μ) MinMax [0,100]
```

## 7. AI 활용 한계 및 보완 사항

**한계 ①:** AI가 생성한 `fallback=2.7`(경남 평균 숙박일수) 같은 도메인 지식 기반 대체값은 실제 통계 검증이 필요하다. 공식 데이터로 검증한 뒤 값을 업데이트한다.

**한계 ②:** CSI 산출식의 가중치(0.40 / 0.25 / 0.20 / 0.15)는 AI가 검증한 것이 아니라 사전 기획값이다. 전문가 검토 또는 민감도 분석(Sensitivity Analysis)을 통해 가중치 적절성을 추가 검증한다.
