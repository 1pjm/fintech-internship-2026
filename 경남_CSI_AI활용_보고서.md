## 1. 개요

경상남도 22개 시군구의 상권 위험도를 정량화하는 **상권 생존지수(CSI)** 산출 및 기부금 배분 의사결정 시스템 구축 프로젝트에서 **Claude AI Agent** 를 핵심 개발 도구로 활용한다. AI는 단순 코드 생성을 넘어 데이터 품질 진단, 산출 논리 검증, 오류 원인 추적까지 분석 전 과정에 개입한다.

---

## 2. AI 활용 영역

### ① 데이터 전처리 파이프라인 설계

9개 원본 데이터를 단일 파일로 병합하려는 초기 설계를 AI가 데이터 입도(Granularity) 분석으로 문제를 사전 식별하고, 카테시안 곱(Cartesian Product) 폭발을 방지하기 위해 **4개 마스터 테이블 구조로 아키텍처를 재설계**한다.

전처리 과정에서 발생한 기술 이슈들에 대해 AI가 원인 진단과 해결 코드를 동시에 제공한다.

| 이슈 | AI 해결 방법 |
|------|------------|
| 인코딩 충돌 (utf-8/cp949 혼재) | 확장자 기반 자동 분기 로드 함수 생성 |
| 날짜 포맷 이종 (`202404`, `2024-04-01` 등) | 정규표현식 기반 `standardize_year_month()` 생성 |
| Mac 한글 파일명 NFD 분해 | `unicodedata.normalize('NFC')` 기반 절대경로 탐색 함수 생성 |
| 병합 키 타입 불일치 (float64 ↔ object) | `fillna('').astype(str)` 전처리 함수 생성 |

### ② EDA 자동화 (코드 생성 → 실행 → 리포트)

AI가 데이터 구조를 분석하여 분석 체크리스트를 수립하고, 이를 **셀 단위 EDA 코드(총 28개 셀 × 2개 파일)** 로 자동 생성한다. 생성된 `.py` 파일을 JSON 파싱으로 `.ipynb` 형식(Google Colab 노트북)으로 자동 변환한다.

EDA 실행 중 발생한 오류에 대해 AI가 원인을 추적하고 수정 코드를 제공한다.

- **matplotlib 한글 폰트 누락:** 폰트 캐시 삭제 → 강제 재로드 → `addfont()` 등록 3단계 해결
- **FixedLocator 불일치 오류:** 데이터에 존재하는 월(6개)과 레이블 수(12개) 불일치를 `.reindex(range(1,13), fill_value=0)` 로 해결
- **NaN 전파로 인한 빈 차트:** LEFT JOIN 후 전체 NaN 컬럼에서 `median()` = NaN → 점수 전체 NaN 연쇄 발생을 추적하고, 도메인 지식 기반 대체값(`fallback=2.7`)을 적용하는 `minmax_safe()` 함수를 설계한다

EDA 완료 후 AI가 코드를 직접 실행하여 차트 10개를 PNG로 저장하고, 각 차트별 분석 해석과 CSI 활용 방안을 포함한 **완전 자급형 HTML 분석 리포트(1.45MB)** 를 자동 생성한다.

### ③ CSI 산출식 논리 검증

초기 CSI 산출식의 치명적 오류 3가지를 AI가 탐지한다.

```
[초기 산출식 — 오류 포함]
CSI = 100 − (매출증감지수 × 0.40 + 업종집중도지수 × 0.25
           + 폐업증가지수 × 0.20 + 소비변동성지수 × 0.15)
```

1. **매출증감지수 방향 역전:** 매출이 증가(+)하면 CSI가 낮아지는 논리 역전 발생 → `매출추세지수`(하락할수록 높아지는 방향)로 수정
2. **폐업증가지수 데이터 부재:** 4개 파일 전체 스캔 결과 폐업 이력 컬럼 부재 확인 → 가맹점 밀도 역수로 대체 권고
3. **연간 비교 스케일 이상:** 전 지역 2023→2024 매출 증가율이 +400%~+1300%로 비정상임을 탐지 → 연도 내 분기별 선형 추세 기울기 방식으로 대체 설계

```
[수정 산출식]
CSI = 100 − (매출추세지수 × 0.40 + 업종집중도지수 × 0.25
           + 상권위축지수 × 0.20 + 소비변동성지수 × 0.15)
```
