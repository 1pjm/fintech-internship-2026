# 기부금 배분 시뮬레이터 변경 리포트 (v4)

> 적용 파일: `csi_dashboard_v3_final_ai.html` — 정책 의사결정(경상남도청) 탭 > 기부금 배분 시뮬레이션 카드

---

## 1. 버전별 공식 진화

| 버전 | 공식 | 핵심 변화 |
|------|------|-----------|
| **v1** | `max(0, 100 − CSI)` | CSI 역비례만 반영 |
| **v2** | `max(0, 100 − CSI) × 인구` | 인구 규모 추가 |
| **v3** | `(100 − CSI)² × 인구` + CSI ≥ 65 제외 | 제곱 강화 + 임계값 컷오프 |
| **v4** ← 현재 | `(100 − CSI)² × 인구 × taper(CSI)` | **완충 구간(50~65) 추가** |

---

## 2. v4 최종 공식

```
가중치ᵢ = (100 − CSIᵢ)² × 유동인구ᵢ × taper(CSIᵢ)

taper(CSI) =
  CSI < 50  →  1.0               (전액 구간 — 완전 적용)
  50 ≤ CSI < 65  →  (65 − CSI) / 15   (완충 구간 — 선형 감쇠)
  CSI ≥ 65  →  0                 (제외 구간 — 배분 없음)

배분 비중 = 가중치ᵢ / Σ가중치 × 100 (%)
배분 금액 = 가중치ᵢ / Σ가중치 × 총예산
```

### 완충 구간 taper 값 예시

| CSI | taper | 설명 |
|-----|-------|------|
| 49.9 | **1.00** (100%) | 전액 구간 직전 |
| 50.0 | **1.00** (100%) | 완충 구간 시작 |
| 52.5 | **0.83** (83%) | 완충 초반 |
| 55.0 | **0.67** (67%) | 완충 중간 |
| 57.5 | **0.50** (50%) | 완충 중간 |
| 60.0 | **0.33** (33%) | 완충 후반 |
| 62.5 | **0.17** (17%) | 완충 후반 |
| 64.9 | **0.01** (1%) | 제외 직전, 극소량 배분 |
| 65.0 | **0.00** (0%) | 제외 구간 시작 |

> v3의 문제: CSI 64.9와 65.1이 전액 vs 0원으로 급격히 갈렸음  
> v4의 해결: 50~65 구간(15점 폭)에서 비중이 서서히 0으로 수렴, 경계값 충격 완화

## 5. 트레이드오프

### v4의 장점
- **경계값 충격 제거**: CSI 64.9 ↔ 65.1 전액/0 절벽이 사라지고 50~65 구간이 완만하게 수렴
- **정책 설명 용이**: "50점 미만 전액, 50~65점 비례 감소, 65점 이상 제외"로 직관적 설명 가능
- **제곱 비선형 유지**: 최하위 위험지역(남해·산청) 집중 배분 효과 유지

### v4의 한계
- 완충 구간 기준(50, 65)은 임의 설정값 — 정책 목적에 따라 조정 필요
- 현재 데이터에서 완충 구간 해당 지역은 합천군·고성군·함안군 3곳
- 인구 역설 일부 잔존: 함안군(CSI 56.2, 인구 46만, 88%)이 의령군(CSI 45.9, 인구 17만, 100%)보다 여전히 더 많이 받음

## 6. 코드 변경 내역

### 완충 함수 추가

```javascript
function budgetTaper(csi) {
  if (csi >= 65) return 0;
  if (csi >= 50) return (65 - csi) / 15;  // 선형 감쇠 (50~65)
  return 1;
}
```

### 가중치 계산 변경

```diff
- const weights = sorted.map(([name, d]) => {
-   if (d.csi >= 65) return 0;
-   const gap = 100 - d.csi;
-   return gap * gap * (POPULATION[name] || 1);
- });

+ const weights = sorted.map(([name, d]) => {
+   const t = budgetTaper(d.csi);
+   if (t === 0) return 0;
+   const gap = 100 - d.csi;
+   return gap * gap * (POPULATION[name] || 1) * t;
+ });
```

### 테이블 변경
- 컬럼 추가: **적용 구간** (전액 / 완충 xx% / 제외)
- 완충 구간: 주황색 `완충 88%` 형태로 표시
- 전액 구간: 초록색 `전액`
- 제외 구간: 회색 `제외`, 행 투명도 40%

## 7. 전체 공식 구조 최종 정리

```
배분 금액ᵢ = (100 − CSIᵢ)²  ×  유동인구ᵢ  ×  taper(CSIᵢ)
              └─ 취약성 강조 ─┘  └─ 규모 ─┘  └─── 구간 보정 ───┘

             / 1.0            (CSI < 50 : 전액)
taper(CSI) = | (65−CSI)/15   (50 ≤ CSI < 65 : 완충)
             \ 0              (CSI ≥ 65 : 제외)
```
