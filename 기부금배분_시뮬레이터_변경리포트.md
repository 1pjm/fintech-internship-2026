# 기부금 배분 시뮬레이터 변경 리포트 (v3)

> 적용 파일: `csi_dashboard_v3_final_ai.html` — 정책 의사결정(경상남도청) 탭 > 기부금 배분 시뮬레이션 카드

---

## 1. 버전별 공식 진화

| 버전 | 공식 | 핵심 변화 |
|------|------|-----------|
| **v1** (최초) | `max(0, 100 − CSI)` | CSI 역비례만 반영 |
| **v2** (1차 수정) | `max(0, 100 − CSI) × 인구` | 인구 규모 추가 |
| **v3** (현재) | `(100 − CSI)² × 인구` + CSI ≥ 65 제외 | 취약성 제곱 강화 + 임계값 컷오프 |

---

## 2. 현재(v3) 배분 공식 상세

```
[ CSI < 65 인 지역 ]
가중치ᵢ = (100 − CSIᵢ)² × 유동인구ᵢ

[ CSI ≥ 65 인 지역 ]
가중치ᵢ = 0  →  배분 제외

배분 비중 = 가중치ᵢ / Σ가중치 × 100 (%)
배분 금액 = 가중치ᵢ / Σ가중치 × 총예산
```

### 적용된 두 가지 방안

#### ① 취약성 제곱 강화

| 구분 | 공식 | 효과 |
|------|------|------|
| v2 (선형) | `(100 − CSI)` | CSI 10점 차이 → 가중치 10 차이 |
| v3 (제곱) | `(100 − CSI)²` | CSI 10점 차이 → 가중치 차이가 훨씬 커짐 |

예시: CSI 20 vs CSI 60
- 선형: 80 vs 40 → **2배** 차이
- 제곱: 6400 vs 1600 → **4배** 차이

→ CSI가 낮을수록 훨씬 더 강하게 우선순위가 높아지는 비선형 구조

#### ② CSI ≥ 65 지역 배분 제외

- 보통(50~64점)·안정(65점↑) 구간 중 **안정 구간은 지원 대상에서 제외**
- 해당 지역: 창녕군·거창군·거제시·사천시·창원시 마산회원구·통영시·창원시 마산합포구·밀양시·창원시 진해구·김해시·양산시·창원시 성산구·창원시 의창구·진주시 (총 **14개 지역**)
- 배분 집중 지역: CSI < 65인 **8개 위험·주의·보통 지역**

## 5. 세 버전 배분 방향 비교 요약

```
v1 (CSI 역비례만)
  → 위험 소규모 군에 집중, 규모 무시
  → 남해군 10.86억, 산청군 9.89억 / 진주시 1.99억, 김해시 2.27억

v2 (CSI × 인구)
  → 규모 반영으로 대도시 배분 증가, 소규모 군 감소
  → 거제시 11.41억, 김해시 10.91억 / 산청군 3.53억, 의령군 2.20억

v3 (CSI² × 인구, CSI<65만)
  → 위험지역에 강하게 집중 + 안정지역은 완전 제외
  → 남해군 26.38억, 산청군 15.74억 / CSI≥65 지역 전부 0억
  → 100억 전액이 8개 위험·주의·보통 지역으로만 배분
```

## 6. 트레이드오프

### v3의 장점
- **집중도 극대화**: 100억 전액이 실제 위험 지역 8곳으로만 배분
- **비선형 강화**: 제곱 적용으로 CSI 최하위(남해·산청)에 훨씬 강한 우선순위
- **명확한 기준선**: CSI 65점이라는 객관적 컷오프로 정책 설명이 용이

### v3의 한계
- **안정 지역 완전 소외**: CSI 65 이상 14개 지역(거제·김해·창원 등 대부분의 대도시)은 한 푼도 받지 못함
- **인구 역설**: 제곱이 취약성을 키우지만 인구 승수 때문에 함안군(CSI 56.2, 인구 46만)이 의령군(CSI 45.9, 인구 17만)보다 더 많이 받는 현상 존재
- **경계값 민감성**: CSI 64.9와 65.1의 차이가 전액 vs 0원으로 벌어짐 → 완충 구간 설계 고려 가능

## 7. 코드 변경 내역

```diff
- // (100-CSI) × 인구
- const weights = sorted.map(([name, d]) =>
-   Math.max(0, 100 - d.csi) * (POPULATION[name] || 1)
- );

+ // (100-CSI)² × 인구, CSI≥65 제외
+ const weights = sorted.map(([name, d]) => {
+   if (d.csi >= 65) return 0;
+   const gap = 100 - d.csi;
+   return gap * gap * (POPULATION[name] || 1);
+ });
```

테이블 표시:
- CSI ≥ 65 지역: 행 투명도 45%, 순위·배분비중·금액 모두 `—` 표시
- CSI < 65 지역: 정상 표시, 순위는 배분 대상 내 순위
