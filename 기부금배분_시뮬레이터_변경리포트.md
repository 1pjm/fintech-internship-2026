> 적용 파일: `csi_dashboard_v3_final_ai.html` — 정책 의사결정(경상남도청) 탭 > 기부금 배분 시뮬레이션 카드

## 1. 변경 요약

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 테이블 컬럼 | 순위 / 지역 / CSI 점수 / 위험 등급 / 배분 비중 / 배분 금액 | + **유동인구(만명)**, **상권 수** 컬럼 추가 |
| 배분 가중치 공식 | `max(0, 100 − CSI)` | `max(0, 100 − CSI) × 유동인구` |
| 카드 부제목 | CSI 역비례 가중 배분 제안 | CSI 역비례 × 인구규모 가중 배분 제안 |
| 데이터 소스 | CSI 점수만 사용 | CSI + 분기별 유동인구(연간 평균) + 시군구 점포수 |

## 2. 배분 공식 비교

### 변경 전 — CSI 단독 역비례

```
가중치ᵢ = max(0, 100 − CSIᵢ)
배분 비중 = 가중치ᵢ / Σ가중치 × 100 (%)
배분 금액 = 가중치ᵢ / Σ가중치 × 총예산
```

- CSI가 낮을수록(위험할수록) 가중치가 커짐
- 지역 규모(인구·상권 수)는 전혀 반영되지 않음
- 결과: 소규모 농촌 군에 예산이 집중

### 변경 후 — CSI 역비례 × 인구 규모

```
가중치ᵢ = max(0, 100 − CSIᵢ) × 유동인구ᵢ
배분 비중 = 가중치ᵢ / Σ가중치 × 100 (%)
배분 금액 = 가중치ᵢ / Σ가중치 × 총예산
```

- **취약성(CSI 낮을수록) × 규모(유동인구 많을수록)** 동시 반영
- 상권 수는 표 표시 전용 (CSI와 중복 가산 방지)

### 설계 의도

| 변수 | 방향 | 역할 | 이유 |
|------|------|------|------|
| CSI | 낮을수록 ↑ | **취약성** 지표 | 위험 지역 우선 지원 |
| 유동인구 | 많을수록 ↑ | **규모** 스케일 | 실제 수혜 대상(점포·소비자) 비례 |
| 상권 수 | 표시만 | 참고 정보 | CSI와 중복 계산 방지 |

> 상권 수를 계산에 넣지 않은 이유: CSI 자체가 이미 업종집중(HHI)·상권위축·매출감소를 반영한 지수이므로, 상권 수를 추가하면 취약 소규모 지역에 같은 취약성을 이중 가산하게 됨.

## 5. 트레이드오프 및 한계

### 신 공식의 장점
- **실효성**: 예산이 실제 점포·소비자가 많은 지역에 비례 배분됨
- **이중 계산 방지**: 상권 취약성(CSI) + 규모(인구)를 직교 변수로 분리
- **직관성**: "얼마나 어려운가 × 얼마나 큰가" 로 해석 가능

### 신 공식의 한계
- CSI가 좋더라도 인구가 많으면 배분 증가 → 순수 취약 지원 성격 약화
- 소규모 농촌 군(남해·산청·의령 등)의 배분이 크게 줄어 형평성 논란 가능
- 유동인구 데이터가 실거주 인구와 다를 수 있음 (관광지 등 변동성 존재)

### 대안 확장 방안
```
# 취약성 가중을 강화하려면 제곱 적용
가중치 = (100 − CSI)² × 인구

# 일정 점수 이상 지역 제외 (강한 선별 지원)
가중치 = (100 − CSI) × 인구  [CSI ≥ 65인 지역은 0으로 처리]

# 가중치 슬라이더 UI로 α 조정 가능하게
가중치 = (100 − CSI)^α × 인구  (α = 1~2 조정)
```

## 6. 코드 변경 내역

```diff
- const weights = sorted.map(([, d]) => Math.max(0, 100 - d.csi));
+ const weights = sorted.map(([name, d]) => Math.max(0, 100 - d.csi) * (POPULATION[name] || 1));

- return `<tr><td>${i+1}</td><td>${name}</td>
-   <td>${d.csi}</td><td>${badge}</td>
-   <td>${pct}%</td><td>${amt}</td></tr>`;
+ return `<tr><td>${i+1}</td><td>${name}</td>
+   <td>${d.csi}</td><td>${badge}</td>
+   <td>${pop}만명</td><td>${stores}개</td>
+   <td>${pct}%</td><td>${amt}</td></tr>`;
```

추가된 상수 객체:
- `POPULATION` — 22개 시군구 유동인구 연간 평균값
- `STORES` — 22개 시군구 점포 수
